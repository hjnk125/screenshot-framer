# Screenshot Framer

A free web tool to wrap screenshots in device or browser frames and export as PNG  
스크린샷을 깔끔한 디바이스·브라우저 프레임으로 감싸 PNG로 저장하는 무료 웹 도구

[![Live](https://img.shields.io/badge/Live-screenshot--framer.vercel.app-black?style=for-the-badge&logo=vercel)](https://screenshot-framer.vercel.app/)

![Screenshot Framer](https://screenshot-framer.vercel.app/og-image.png)

---

## Who it's for / 이런 분께 유용해요

- Sharing app or web screenshots on social media or in a portfolio  
  앱·웹 스크린샷을 SNS나 포트폴리오에 예쁘게 올리고 싶을 때
- Creating product promo images without a design tool  
  제품 소개 이미지를 빠르게 만들고 싶을 때
- Getting a quick mockup without opening Figma  
  Figma 없이 목업 이미지가 필요할 때

## Features / 주요 기능

|                                                |                                                                                     |
| ---------------------------------------------- | ----------------------------------------------------------------------------------- |
| 🖥️ **Device frames / 디바이스 프레임**         | MacBook Air/Pro, MacBook Neo, iMac 24, iPhone 15/16/17 (Air, Pro, Pro Max)          |
| 🌐 **Browser frames / 브라우저 프레임**        | Chrome, Safari Big Sur/Catalina — light & dark themes / 라이트·다크 테마            |
| ✏️ **Browser customization / 브라우저 커스텀** | Set URL, tab title, and favicon / URL, 탭 타이틀, Favicon 직접 설정                 |
| 🎨 **Background / 배경 옵션**                  | Transparent, white, black, or custom image / 투명·흰색·검정·커스텀 이미지           |
| 💡 **Shadow / 그림자 조절**                    | Toggle opacity. Off = zero padding / On/Off + 불투명도. Off 시 여백 없이 export     |
| 🔍 **Scale & pan / 스케일 & 패닝**             | Slider to resize, drag to reposition / 슬라이더로 크기 조정, 캔버스에서 직접 드래그 |

## How to use / 사용 방법

1. Upload a screenshot (PNG or JPG) / 스크린샷 파일을 올린다 (PNG·JPG)
2. Pick a device or browser frame / 원하는 디바이스 또는 브라우저 프레임을 고른다
3. Adjust the position and scale / 이미지 위치와 크기를 조정한다
4. Set background and shadow (optional) / 배경·그림자 옵션을 설정한다 (선택)
5. Hit **Export PNG** / **Export PNG** 버튼을 누른다
